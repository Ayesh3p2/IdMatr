Deployment

Deploy in 5 Minutes

Single docker compose command. On-premises or any cloud provider.

git clone https://github.com/your-org/idmatr
cp .env.example .env
# 2. Set your secrets in .env
JWT_SECRET=$(openssl rand -base64 64)

# 3. Start everything
docker compose up -d
